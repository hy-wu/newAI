## 1. Tensor Network + LLM / Attention

### 現狀

2024–2025 年 tensor network 與 transformer 的交叉主要集中在**壓縮**和**分析**兩個方向，而非架構設計。

### 關鍵論文

**TensorLLM — Tensorising Multi-Head Attention** (IJCNN 2025)
- 用 Tucker 分解對 MHA 權重做張量化，實現 ~250× 壓縮
- 不需重新訓練，可與 FFN 去噪結合
- 壓縮後推理能力反而提升（推測是去噪效應）
- Code: https://github.com/guyuxuan9/TensorLLM

**DisCoCLIP — MPS Encoder for Vision-Language** (2025)
- 用 Matrix Product States (MPS) 取代 CLIP 的 text encoder
- CCG 句法分析 → 詞張量按語法推導做收縮
- 參數從數千萬降到 <100 萬
- 四種 tensor network 拓撲比較：Tree, Compact, Cups (TT), Spider

**MPS for Simultaneous Classification and Generation** (arXiv:2406.17441, 2024)
- MPS 同時做分類器和生成器，受 GAN 啟發

### 空白

- **DMRG sweeping 作為推理算法**：目前沒人做。DMRG 的雙向掃描策略（left-to-right → right-to-left → converge）可以對應 LLM 的 iterative refinement 推理
- **MPS 作為 token 表徵而非權重壓縮**：現有工作全是壓縮權重矩陣，沒人把 token 序列本身當 MPS 來處理
- **TEBD (Time-Evolving Block Decimation) 對應 transformer layer**：time evolution 的 Suzuki-Trotter 分解可以對應多頭注意力的分解

## 2. Higher-Rank Tensor Attention

### 現狀

標準 attention 是 rank-3 運算（query, key, value 三個索引）。你筆記中 `rank 3 → 4` 的精確方向是 2025–2026 的熱點。**你下載的 Poly-Attention 論文（2602.02422，Columbia University，Feb 2026）是這個方向的里程碑。**

### 關鍵論文（你下載的）

**Poly-Attention: A General Scheme for Higher-Order Self-Attention** (arXiv:2602.02422, Feb 2026)
- **你下載的這篇是核心文獻。**
- 定義了統一的 higher-order attention 框架（poly-attention），以**注意力多項式 (attention polynomial)** 來刻畫任意高階注意力機制
- 證明標準 self-attention 只能做 pairwise，**連 2-fold function composition（如「A 住在 B 城，B 城在 C 國，A 在哪國？」）都做不了**
- 給出完整的複雜度-表達力 trade-off 圖景：

| 機制 | 精確計算 | 近似計算 | 能做 function comp? |
|------|---------|---------|-------------------|
| Self-attention | O(n²) | O(n¹⁺ᵒ⁽¹⁾) | ❌ 連 2-fold 都不行 |
| 3-Tensor attention | O(n³⁺ᵒ⁽¹⁾) | O(n¹⁺ᵒ⁽¹⁾) | ✅ 2-fold |
| Strassen attention | O(n^ω) ω≈2.37 | O(n¹⁺ᵒ⁽¹⁾) | ✅ 2-fold |
| **Tree-attention (新建議)** | **O(n²)** | **O(n¹⁺ᵒ⁽¹⁾)** | **✅任意 r-fold** |

- **Tree-attention** 是他們提出的最佳權衡：quadratic 時間 + 能做任意深度的 function composition
- 嚴格證明了 lower bound：某些 higher-order attention 不可能比 O(n²) 更快（基於 fine-grained complexity 假設）
- 附帶實驗驗證（Section 5）
- **直接回答了你的「rank 3 → 4」問題**——Poly-Attention 框架涵蓋了任意 order t 的注意力，且 tree-attention 在 quadratic 時間內達成了之前需要 super-quadratic 才能做的事

**Deep Polynomial Neural Networks (π-Nets)** (arXiv:2006.13026, 2021)
- 你下載了這篇的 v2。輸出是輸入的**高階多項式**，參數用**張量分解 + 因子共享**來避免組合爆炸
- 三種分解（CP、Tucker、Hierarchical Tucker）對應不同的 hierarchical 網路結構
- 即使**不使用非線性激活函數**也能產生好結果——說明高階多項式本身就有足夠表達力
- 在圖像生成、人臉驗證、3D mesh 上達到 SOTA

### 其他關鍵論文

**Higher-Order Transformers (HOT)** (arXiv:2412.02919, 2024)
- Kronecker 分解的注意力機制，處理 rank ≥ 3 的張量數據（多變量時序、3D 醫學影像）
- 複雜度從 O(N³) 降到每個軸的 O(N_axis²)

**Tensor Attention Training** (arXiv:2405.16411, 2024)
- 理論證明 tensor attention 的反向傳播可以在 O(n^(1+o(1))) 完成（跟 forward 同量級）

**On the Benefits of Rank in Attention Layers** (arXiv:2407.16153, 2024)
- 單個 full-rank attention head 可以表示低秩 attention 無法逼近的函數（除非 head 數指數增長）
- **直接支持你的「雙層化以增強 token 佔比差異」方向**

### 空白（更新版）

- **Tree-attention 的實際實現**：Poly-Attention 論文給出了理論和 toy experiment，但還沒有大規模 LLM 的實現
- **你的 rank-3 → rank-4 仍然成立**：Poly-Attention 處理的是 attention 的階數（order t），而你說的 rank 4 是張量的秩。兩者是正交的——你可以有高階 + 高秩的注意力
- **物理類比**：rank-2 是兩體相互作用（庫倫力），rank-3 是三體，rank-4 是四體。Poly-Attention 說的是「允許幾體參與交互」，rank 說的是「交互的解析度有多高」——兩者結合才有完整的物理圖像

## 3. Clifford / Geometric Algebra in Deep Learning

### 現狀

這是 2024–2025 增長最快的方向之一，但主要集中在**等變神經網路 (Equivariant NN)**，沒人用於 LLM 的 token 表徵（除了 BIIC）。

### 關鍵論文

**Clifford Group Equivariant Neural Networks** (NeurIPS 2023 Oral)
- 奠基之作，證明 Clifford group action 可以在所有 multivector grade 上實現等變性
- 2024–2025 大量工作在此基礎上擴展

**GLGENN** (ICML 2025 & IJCNN 2025)
- 基於 Clifford 廣義 Lipschitz 群等變網路
- **參數極少**，在物理、視覺、幾何回歸任務上超越 baselines

**STAResNet — Spacetime Algebra ResNet** (arXiv:2408.13619, 2024)
- 在 Spacetime Algebra 中構建 ResNet 求解 Maxwell PDE
- MSE 比標準 Clifford ResNet 低 2.6×，參數少 6×
- **關鍵教訓**：代數的選擇（Cl(3,1) vs Cl(4,1) vs 其他）對性能有巨大影響

**Randomized Geometric Algebra for Convex Neural Networks** (arXiv:2406.02806, 2024)
- 隨機線性代數推廣到 Clifford 代數 → 全局最優凸訓練
- 已應用於 GPT-4 / BERT embedding 微調，比 AdamW 更快更穩健
- **重要性**：證明 Clifford 代數可以用於 LLM 相關任務

**From Complexity to Clarity** (ICLR 2025, Mert Pilanci)
- 證明深度 ReLU 網路的最優權重可以用 Clifford 代數中的 **wedge product** 寫出閉式表達
- 訓練等價於 signed volumes 的凸優化

### 空白

- **Clifford 做 token 表徵而非層間變換**：只有 BIIC 在做
- **SlowFast Clifford 架構**：BIIC 規劃但未實現
- **Clifford + attention 的融合**：GATr 用 Clifford 替換了 attention 的內部計算，但 token 本身仍是向量

## 4. Feynman Diagram + Neural Network

### 現狀

這個方向有兩條路線：用 Feynman diagram **分析**神經網路（NTK 微擾展開），或用 QFT **設計**神經網路架構。

### 關鍵論文

**NCoder — QFT Approach to Encoding Data** (arXiv:2402.00944, 2024)
- 自編碼器的 latent layer 表示為 **n-point correlation functions**
- 圖像視為 lattice field theory 的樣本，用 Feynman diagram 構建有效作用量
- MNIST 上僅用前三階關聯函數即可正確分類
- **最接近你「redefine AI infra with Feynman diagram」方向的論文**

**Feynman Diagrams as Computational Graphs** (arXiv:2403.18840, 2024)
- 用 Dyson-Schwinger + parquet 方程的高階 Feynman diagram 表示為計算圖
- 使用 ML framework 的 Taylor-mode 自動微分來評估
- **關鍵 insight**：Feynman diagram 本身就是 computational graph——節點是交互頂點，邊是傳播子

**Finite-Width NTK from Feynman Diagrams** (Guillen et al., 2025)
- 用 Feynman diagram 計算 NTK 的有限寬度修正
- ReLU 對角線 Gram 矩陣無有限寬度修正——這是 diagrammatic 方法的洞察

**Neural Network Perturbation Theory via Born Series** (arXiv:2009.03192, 持續引用)
- MLP 的 propagator + vertex 類比
- 散射長度計算 + machine-learned Born 近似

### 空白

- **沒有用 Feynman diagram 設計 transformer 架構的工作**。NCoder 是自編碼器，不是 LLM。
- **對應關係**：你的筆記點出了 external line → token, vertex → attention, loop → recurrence，這套對應如果形式化可以產出一個新架構
- 具體可做的：**用 diagrammatic 展開來決定網路需要多少層**（哪個 loop order 對應哪種推理能力）

## 5. p-adic Numbers + Machine Learning

### 現狀

2024–2025 是 p-adic ML 的爆發期。你筆記中「語言的樹形層級在這個數系裡天然精確」已被驗證。

### 關鍵論文

**v-PuNNs: van der Put Neural Networks** (arXiv:2508.01010, Aug 2025)
- **最重要的進展**。第一個神經元是 ℤₚ 中 p-adic ball 的示性函數的網路
- TURL (Transparent Ultrametric Representation Learning)：每個權重對應一棵明確的子樹
- WordNet 名詞（52,427 葉節點）**99.96% 準確率**，Gene Ontology **100%**
- CPU 訓練，參數遠少於 transformer / hyperbolic 模型

**Learning with the p-adics** (arXiv:2512.22692, Dec 2025, André F. T. Martins)
- 建立 ℚₚ 上的分類、回歸、表徵學習理論框架
- 證明 Quillian 語義網路可以表示為**緊湊的 p-adic 線性網路**——實數做不到

**Ultrametric Attention** (GitHub: Dicklesworthstone/model_guided_research, 2025)
- JAX + PyTorch 實現，O(N log N) 複雜度
- 使用 bit-prefix LSH signatures + tree-structured addressing
- **直接替代標準 softmax attention**

**Adelic Operation-preserved Embeddings (AOE)** (Korea University, Sep 2025)
- 免訓練的數值表徵，結合實數 + p-adic 分量
- 可直接插入 transformer 提升數學推理能力

**p-adic Linear Regression** (arXiv:2510.00043, Sep 2025)
- 在 p-adic 空間中最小化距離的最優平面必須經過至少 n+1 個數據點——跟 Euclidean 完全不同
- 應用於 NLP：層級分類學 + 語法形態學

### 空白

- **p-adic + Clifford 的混合表徵**：還沒人做。p-adic 提供骨架（離散樹），Clifford 提供血肉（連續幾何變換）
- **p-adic position encoding**：RoPE 用旋轉編碼位置，p-adic 可以用樹深度編碼位置，兩者可以結合
- **Trainable p-adic attention**：現有的 ultrametric attention 用固定 p-adic 度量，可學習的 p-adic 度量會更靈活

## 6. Density Matrix + LLM / Attention

### 現狀

量子機率形式用於 attention 在 2024–2025 有穩步進展。你筆記中的「多義性變成疊加態，語境決定坍縮方向」已有雛形實現。

### 關鍵論文

**QSIM — Quantum-inspired Hierarchical Semantic Interaction** (Neurocomputing, Jan 2025)
- 用 Hilbert space 中的 **reduced density matrix** 表示詞表徵
- Schmidt 分解提取「核心語義空間」，entanglement entropy 度量詞級語義交互
- 15 個文本分類數據集上超越經典神經網路和 prior quantum 語言模型

**Attention Density Matrix (ADM)**
- Attention 分佈表示為量子 density matrix，而非逐點獨立機率
- **非對角元素**編碼 token 對之間的不確定性與成對依賴——經典 attention 做不到
- 應用於 seq2seq, 機器翻譯

**Quantum LLMs via Tensor Network Disentanglers** (arXiv:2410.17397, Oct 2024)
- 用 MPO (Matrix Product Operator) 分解 self-attention + MLP 的權重矩陣
- 低記憶體量子增強 LLM

**BERT-Residual Quantum Language Model** (IEEE Access, Jan 2025)
- 整合 density matrix 建模到 BERT fine-tuning
- ODE 多步法融合經典 BERT 表徵 + density matrix 高階語義交互
- 解決 density matrix 加入大模型時的梯度爆炸/消失問題

### 空白

- **Density matrix + Clifford multivector 的等價性**：Cl(4,1) 有 faithful matrix representation in ℂ(8) 或 ℝ(16)，32 維 multivector 可以嵌入為 8×8 密度矩陣。這意味著你的「混合體」可能比你想像的更自然
- **語境坍縮的 learning rule**：目前 QSIM 等的坍縮是固定的（投影測量），沒人做可學習的 context-dependent collapse

## 7. Softmax / Temperature / Statistical Mechanics

### 現狀

這條線在 2024–2025 有多個突破，特別是理解 temperature 的理論角色和 attention 的統計力學類比。

### 關鍵論文

**Attention to Order: Transformers Discover Phase Transitions** (arXiv:2510.07401, Oct 2025)
- **最重要的論文之一**。用 softmax attention transformer 研究 2D Ising model
- Transformer 通過 attention entropy 無監督地恢復了臨界溫度 T_c ≈ 2.27
- 低溫（有序相）→ bimodal attention distribution + 長程關聯
- 高溫（無序相）→ uniform attention
- **直接證明 attention 和 Ising model 同構**

**Optimal Attention Temperature Enhances ICL under Distribution Shift** (Demir & Dogan, Nov 2025)
- 第一個統一的 attention temperature 理論
- 推導 closed-form generalization error 並證明存在最優 τ
- GPT-2 和 LLaMA2-7B 上驗證

**Unpacking Softmax: Temperature Drives Rank Collapse** (arXiv:2506.01562, 2025)
- Temperature 如何影響表徵 rank、壓縮、OOD 泛化

**Softmax is not Enough (for Sharp Size Generalisation)** (Veličković, ICML 2025)
- Softmax attention 隨著輸入增長必然分散
- 提出 adaptive temperature 作為解決方案

**Regularizing the Entropy Landscape of Self-Attention** (NeurIPS 2025)
- 標準訓練在淺層 induce 了 high-entropy attractor
- 可學習的 per-head temperature 可以重塑 entropy 曲線

**Scalable-Softmax (SSMax)** (Nakanishi, Jan 2025)
- 用 n^{s·z_k} 取代 exp(z_k)，n 是序列長度，s 是可學習參數
- 保留了指數族結構但動態適應序列長度

### 空白

- **Temperature field**：目前所有工作都是全局或 per-head temperature，你筆記中的想法可以延伸到 per-position temperature——類似於空間變溫度的 Ising model
- **Softmax 替換為一般指數族分布**：t-distribution 的 heavy tail 可用於控制 attention 的稀疏性，還沒人在 LLM scale 上系統測試
- **Phase transition 作為湧現能力的預測指標**：如果 attention entropy 的相變點可以預測某個能力的湧現，那就可以用來指導訓練

## 8. Moiré / Twisted Angle / 4DGS

### 現狀

**Moiré 物理和深度學習的交叉目前完全不存在。** 你的 insight（轉角結構帶來湧現能力）還是全新的方向。

### 相關但分離的工作

**Twisted Bilayer Graphene (物理)**
- Magic angle 1.1° 時產生超導——兩層平庸材料旋轉後湧現全新物理
- DNN 被用來預測 moiré 超晶格中的聲子、摩擦等性質
- 但沒有人用 twisted angle 的類比來設計神經網路架構

**4D Gaussian Splatting (CV)**
- 4-LEGS (Eurographics 2025)：時空語言特徵嵌入 4D Gaussian
- FluidNexus (CVPR 2025)：4DGS + 物理模擬做流體重建
- 4DGS as Learned Dynamical System (arXiv:2512.19648, 2025)：連續時間動力學

**Moiré 在 CV 中的含義完全不同**
- Image demoiréing（去除相機-顯示器干涉條紋）跟 twistronic moiré 完全無關
- 所以你如果能把 **twisted angle 物理**引入神經網路，這是一個**真實的空白**

### 可做的方向

- **Twisted embedding layers**：兩層 embedding 之間轉一個可學習角度，在 magic angle 附近湧現新能力
- **Spacetime rotation as feature transformation**：不同 token 之間的 SO(3,1) 變換參數化（比點積有更多自由度）
- **4DGS 作為 token 表徵**：每個 token 不是向量，而是 4D Gaussian——位置 + 形狀 + rotation，在時空中做 splatting

## 9. Fluid Dynamics + VLM / World Model

### 現狀

你的筆記「VLM - fluid simulation」——這個對應在 2024–2025 已有多項進展，但大多是 physics-informed，而非架構層面的類比。

### 關鍵論文

**FluidNexus** (CVPR 2025)
- 單視頻 → 3D 流體重建 + 預測
- View-synthesis + video diffusion + Position-Based Fluid + 3DGS
- **最接近 video generation = fluid simulation 的工作**

**How Far is Video Generation from World Model?** (ByteDance, ICML 2025)
- 系統評測 diffusion 視頻生成模型是否符合物理定律
- ✅ In-distribution：完美（scaling 有效）
- ⚠️ Combinatorial：可衡量的 scaling
- ❌ OOD：完全失敗
- 模型表現出 **case-based generalization**（匹配最接近的訓練樣本），而非抽象物理規則
- 引用優先級：color > size > velocity > shape

**PROSE-FD** (NeurIPS 2024)
- 多模態 PDE foundation model，zero-shot 預測流體動力學
- Transformer 架構，覆蓋 Navier-Stokes、shallow water、複雜幾何
- 60K+ 軌跡預訓練

**PINNs4Drops** (2024)
- Video-conditioned PINN 做兩相流（液滴）重建
- 整合實驗觀測 + Navier-Stokes 控制方程

**Synthetic Vision: Training VLMs to Understand Physics** (arXiv:2412.08619, Dec 2024)
- Fine-tune VLM on physics simulation QA
- Physics Context Builders (PCBs)：生成含有物理屬性的場景描述
- 小模型 fine-tune 大幅超過大 foundation model

### 空白

- **架構層面類比**：frame 序列 = Lagrangian 粒子軌跡，frame warp field = velocity field。如果你用 Navier-Stokes 的離散化（如 SPH）來設計 video transformer 的 token 更新規則，這是一個新方向
- **物理守恆量作為架構歸納偏置**：mass/momentum/energy conservation 對應 transformer 的某種守恆量。目前只有散度自由（divergence-free velocity）被使用了

## 10. Entanglement Entropy + Interpretability

### 現狀

量子資訊理論的工具正在被用於理解神經網路的內部表徵。

### 關鍵論文

**Probing Information Distribution via Entropy Analysis** (arXiv:2507.15347, 2025)
- Entropy 分析作為探針工具，量化 GPT-based LLM 中 token-level 的不確定性
- Entropy 模式在不同處理階段的變化揭示模型內部行為

**InfoMat: Information Matrix** (Entropy, Mar 2025)
- Mutual information decomposition for sequential data
- 捕獲 directed information, transfer entropy
- 應用於神經科學、金融、通訊和 ML

**Tensor Networks for Explainable ML** (Neurocomputing, 2025)
- MPS 做非監督聚類，天然提取 Von Neumann Entropy + mutual information
- 方法媲美深度學習（autoencoder, GAN）但提供更豐富的可解釋性

**Quantum-Inspired NNs with Stochastic Dynamics** (EAAI, 2025)
- Von-Neumann entanglement entropy 作為量子啟發神經網路的可解釋性指標
- cross-modal 交互用 open quantum system 建模

**Collective Variables of Neural Networks** (OpenReview, ICLR venue)
- NTK 譜熵跨架構研究（transformer, ResNet, GNN）
- 發現兩種主導學習機制：**information compression**（熵減）和 **structure formation**（熵增）

### 空白

- **Entanglement entropy 作為 attention head 重要性指標**：如果某個 head 的 entanglement entropy 低，可能表示它在做 trivial 的任務——可以用來剪枝
- **Mutual information 作為層間冗餘度量**：層和層之間的 MI 可以用來決定哪裡需要 skip connection

## 11. Renormalization Group + Neural Network

### 現狀

RG 與 NN 的交叉在 2024–2025 開始真正落地，從類比走向算法。

### 關鍵論文

**ROMA: Renormalized Operators with Multiscale Attention** (arXiv:2502.15913, Feb 2025)
- 將 RG 整合到 neural operator 中
- 用 attention 機制建模跨尺度的多尺度交互
- 在 >1M 節點的系統上運行，支援長程交互 + 噪聲數據
- **首次在架構層面整合 RG + attention**

**RG Framework for Scale-Invariant Feature Learning** (AAAI 2025, Caltech)
- 將 layer-wise transformation 視為 RG transformation
- 提出 scale-aware activation functions
- RG flow equations for network parameters
- Fixed points = scale-invariant feature representations

### 空白

- **RG 作為 transformer 的理論框架**：每一層 transformer 其實在做 coarse-graining（attention 聚合資訊 → FFN 非線性變換 → residual 保留細節）。形式化這個類比可以回答「需要多少層」的問題
- **Multi-scale attention 分離**：ROMA 用了 attention，但不同 head 沒有被強制學到不同尺度。可以設計顯式的 scale separation（小尺度 head 看局部，大尺度 head 看全局）

## 12. Cognitive Bias + LLM Architecture

### 現狀

目前的工作是**分析 LLM 中出現的認知偏見**，而非**把認知偏見設計進架構**。

### 關鍵論文

**It's All Connected: Miras Framework** (arXiv:2504.13173, Apr 2025)
- 直接用 attentional bias（人類認知傾向）重新審視 transformer + 現代 RNN
- 提出 Moneta, Yaad, Memora 三個序列模型
- 在語言建模和 recall-intensive 任務上超過傳統 transformer
- **最接近你的方向的工作**

**Lost in the Middle: Emergent Property from IR Demands** (OpenReview, 2025)
- LLM 的 U 形性能曲線（primacy/recency effects）類比人類記憶偏見
- 源於訓練數據的需求衝突：某些任務需要均勻回憶（長期記憶），另一些需要最近優先（短期記憶）

**Emergence of Episodic Memory in Transformers** (NAACL 2025)
- GPT-2 attention head 中發現時序偏見：時間連續性、primacy/recency effects
- 由 induction heads 驅動——ablation 消除效果

**Robopsychology — Cognitive Bias Testing in LLMs** (HF Discussion, 2025)
- 系統測試 anchoring, availability, confirmation bias
- 發現 Claude 存在系統性空間扭曲錯誤

### 空白

- **認知偏見作為架構設計原則**：現有工作都是分析 LLM 中「意外」出現的偏見，沒人主動把 anchoring（注意力偏置）、loss aversion（非對稱 activation）、framing effects（上下文窗口）設計進去
- **關鍵問題**：人類認知的 lossy compression 讓人能泛化——那 LLM 的 information bottleneck 是不是不該被當 bug，而是 feature？
